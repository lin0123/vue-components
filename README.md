#CommonMovePopup.vue组件
说明: 只基于vue的pc端弹出菜单可移动组件, 当在弹出框内点击, 可随着鼠标的移动而移动.
参数说明:
    v-model: boolean 类型, 设置为true显示, false隐藏
    selfClass: String 类型, 自定义弹出框样式
    插槽name="delete-crross", 自定义删除按钮
    插槽name="title", 自定义名称
    插槽name="content", 自定义弹出框内容
